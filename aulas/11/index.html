Manipulação Básica de Elementos

Objetivos da Aula

  • Alterar textos, classes e atributos de elementos.
  • Utilizar métodos como textContent, setAttribute, classList.

Conteúdo da Aula

  1. Alterando o texto de um elemento

    • Exemplo de Referência:
      JavaScript
      
element.textContent = "Novo Texto";
element.innerHTML = "<strong>Novo Conteúdo HTML</strong>";

  2. Alterando atributos

    • Exemplo de Referência:
      JavaScript
      
element.setAttribute("class", "novaClasse");
element.classList.add("classeAdicional");
element.classList.remove("classeAntiga");
element.classList.toggle("classeAlternada");
    • Exemplo Prático.
      JavaScript
      
<div id="box" class="red-box"></div>
<button onclick="changeColor()">Alterar Cor</button>
<script>
    function changeColor() {
    let box = document.getElementById("box");
    box.classList.toggle("blue-box");
    }
</script>

  3. Alterando estilos diretamente

    • Exemplo de Referência:
      JavaScript
      
element.style.color = "red";
element.style.backgroundColor = "yellow";

Exercício

Manipulalção básica do DOM

  1. Crie um botão que altera a classe de um elemento para aplicar um novo estilo.
  2. Altere o texto de todos os elementos de uma classe específica.
  3. Crie uma interação que destaque um elemento ao passar o mouse sobre ele.
