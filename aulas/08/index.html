Eventos em JavaScript

Tópico: onclick, onmouseover, onkeydown, e manipulação de eventos no DOM.

Objetivos da Aula

  • Entender como capturar e manipular eventos em elementos HTML utilizando JavaScript.
  • Conhecer os principais eventos: onclick, onmouseover, onkeydown.
  • Aprender a criar interações dinâmicas com elementos na página.

Conteúdo da Aula

  1. O que são Eventos?

    • Eventos permitem que o JavaScript reaja às interações do usuário ou a alterações no DOM.
    • Exemplo de eventos comuns:
      • onclick: Ocorre quando o elemento é clicado.
      • onmouseover: Ocorre quando o mouse passa sobre o elemento.
      • onkeydown: Ocorre quando uma tecla é pressionada.

  2. Trabalhando com Eventos

    • Adicionando Eventos Inline
      • Diretamente no HTML.
      • Exemplo:
        JavaScript
        
<button onclick="alert('Botão clicado!')">Clique Aqui</button>
    • Adicionando Eventos pelo JavaScript
      • Exemplo com addEventListener:
        JavaScript
        
const button = document.getElementById("meuBotao");
button.addEventListener("click", () => {
    alert("Botão clicado!");
});

  3. Tipos de Eventos

    • Evento onclick
      • Ação: Quando o usuário clica em um elemento.
      • Exemplo:
        JavaScript
        
<button onclick="exibirMensagem()">Clique</button>
<script>
    function exibirMensagem() {
        alert("Você clicou no botão!");
    }
</script>
    • Evento onmouseover
      • Ação: Quando o mouse passa sobre um elemento.
        JavaScript
        
<p onmouseover="mudarTexto(this)">Passe o mouse aqui</p>
<script>
    function mudarTexto(elemento) {
        elemento.innerText = "Você passou o mouse!";
    }
</script>
    • Evento onkeydown
      • Ação: Quando uma tecla é pressionada.
        JavaScript
        
<input type="text" onkeydown="mostrarTecla(event)">
<script>
    function mostrarTecla(evento) {
        console.log("Tecla pressionada:", evento.key);
    }
</script>
