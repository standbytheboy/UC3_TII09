Estilizando com JavaScript: Manipulação de Estilos e Classes Dinamicamente

Objetivos da Aula

  • Entender como manipular estilos diretamente em elementos com JavaScript.
  • Aprender a adicionar, remover e alternar classes dinamicamente.
  • Criar efeitos visuais básicos utilizando manipulação de estilos e classes.

Conteúdo da Aula

  1. Manipulando Estilos Diretamente

    • A propriedade style permite alterar o estilo de um elemento diretamente:
      JavaScript
      
element.style.color = "red";
element.style.backgroundColor = "yellow";
element.style.fontSize = "20px";
    • Exemplo Prático:
      JavaScript
      
<button id="changeStyleBtn">Alterar Estilo</button>
<div id="box" style="width: 100px; height: 100px; background-color: lightblue;"></div>

<script>
    const button = document.getElementById("changeStyleBtn");
    const box = document.getElementById("box");

    button.addEventListener("click", () => {
    box.style.backgroundColor = "pink";
    box.style.border = "2px solid black";
    box.style.borderRadius = "10px";
    });
</script>

  2. Adicionando e Removendo Classes

    • classList.add("className"): Adiciona uma classe.
    • classList.remove("className"): Remove uma classe.
    • classList.toggle("className"): Adiciona a classe se ela não existir; remove se já existir.
    • Exemplo Prático:
      JavaScript
      
<button id="toggleClassBtn">Alternar Estilo</button>
<div id="box2" class="blue-box"></div>
                                    
<style>
    .blue-box {
    width: 100px;
    height: 100px;
    background-color: blue;
    }
    .red-box {
    background-color: red;
    }
</style>

<script>
    const toggleBtn = document.getElementById("toggleClassBtn");
    const box2 = document.getElementById("box2");

    toggleBtn.addEventListener("click", () => {
    box2.classList.toggle("red-box");
    });
</script>

Exercícios Práticos

  1. Mudar a cor de um botão ao passar o mouse

    Crie um botão que mude de cor ao passar o mouse sobre ele e volte ao original ao remover o mouse.

    HTML
    
<button id="hoverBtn">Passe o Mouse Aqui</button>

<script>
    const hoverBtn = document.getElementById("hoverBtn");

    hoverBtn.addEventListener("mouseover", () => {
    hoverBtn.style.backgroundColor = "green";
    });

    hoverBtn.addEventListener("mouseout", () => {
    hoverBtn.style.backgroundColor = "";
    });
</script>

  2. Ativar e desativar modo noturno

    Crie um botão que alterna entre modo claro e escuro na página.

    HTML
    
<button id="toggleThemeBtn">Alternar Tema</button>

<style>
    body.dark-mode {
    background-color: black;
    color: white;
    }
</style>

<script>
    const toggleThemeBtn = document.getElementById("toggleThemeBtn");

    toggleThemeBtn.addEventListener("click", () => {
    document.body.classList.toggle("dark-mode");
    });
</script>

  3. Simulando um botão de destaque

    Crie um botão que, ao ser clicado, destaque um parágrafo alterando sua borda e cor de fundo.

    HTML
    
<button id="highlightBtn">Destacar Parágrafo</button>
<p id="paragraph">Este é um parágrafo de exemplo.</p>

<script>
    const highlightBtn = document.getElementById("highlightBtn");
    const paragraph = document.getElementById("paragraph");

    highlightBtn.addEventListener("click", () => {
    paragraph.classList.toggle("highlight");
    });
</script>

<style>
    .highlight {
    background-color: yellow;
    border: 2px solid orange;
    }
</style>

Resumo e Dicas

  • Manipular estilos diretamente é útil, mas usar classes é mais eficiente e organizado.
  • Combine JavaScript e CSS para criar efeitos visuais consistentes e reutilizáveis.
  • Sempre use classList para manipular múltiplos elementos de forma dinâmica e limpa.

Exercício

  1. Crie uma lista de itens, onde ao clicar em cada item, ele fique destacado com uma nova cor e fonte.
  2. Adicione um botão de "Redefinir Estilos" que retorna todos os elementos ao estilo original.
