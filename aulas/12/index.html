Criando e Removendo Elementos do DOM

Objetivos da Aula

  • Aprender a criar novos elementos no DOM.
  • Adicionar e remover elementos dinamicamente.

Conteúdo da Aula

  1. Criando um novo elemento

    • Exemplo de Referência:
      JavaScript
      
let newElement = document.createElement("div");
newElement.textContent = "Novo Elemento";
document.body.appendChild(newElement);

  2. Adicionando elementos em uma posição específica

    • Exemplo de Referência:
      JavaScript
      
let parent = document.getElementById("parent");
parent.appendChild(newElement);

  3. Removendo um elemento

    • Exemplo de Referência:
      JavaScript
      
parent.removeChild(newElement);

  1. Exemplos Práticos

    • Adicionar um item em uma lista
      JavaScript
      
<ul id="list">
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
<button onclick="addItem()">Adicionar Item</button>
<script>
function addItem() {
    let list = document.getElementById("list");
    let newItem = document.createElement("li");
    newItem.textContent = "Novo Item";
    list.appendChild(newItem);
}
</script>
    • Remover um item da lista
      JavaScript
      
<button onclick="removeItem()">Remover Último Item</button>
<script>
    function removeItem() {
    let list = document.getElementById("list");
    if (list.lastChild) {
        list.removeChild(list.lastChild);
    }
    }
</script>

Exercício

Manipulalção básica do DOM

  1. Crie um botão que adicione um parágrafo ao final de um div.
  2. Faça um botão que remova o primeiro item de uma lista.
  3. Implemente um campo de texto e botão que insere o conteúdo digitado como um novo elemento na página.
