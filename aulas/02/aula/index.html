Utilizando console.log, innerText e prompt para saída e entrada de dados.

Aprender a exibir dados no console, interagir com o conteúdo da página usando innerText e solicitar entradas de dados com o prompt, aplicando cada um em exemplos práticos.

Introdução e Objetivos

  • Introduzir os conceitos de entrada e saída de dados no contexto do JavaScript para manipulação do DOM e desenvolvimento interativo.
  • Explorar os comandos principais para exibir informações e capturar entradas, focando em três métodos: console.log, innerText e prompt.

Saída de Dados

  1. console.log
    • Permite visualizar informações no console do navegador, útil para depuração e visualização de dados
    • Sintaxe: console.log("Mensagem");
    • Exemplo
      JavaScript
      
console.log("Olá, mundo!");  // Exibe "Olá, mundo!" no console
let nome = "Aecio";
console.log("Bem-vindo, " + nome);  // Exibe "Bem-vindo, Aecio" no console
  2. innerText
    • Modifica o conteúdo de texto de um elemento HTML, alterando o que o usuário vê na página.
    • Sintaxe: document.getElementById("idDoElemento").innerText = "Novo texto";
    • Exemplo
      • JavaScript
      
<div id="mensagem"></div>
<script>
  let nome = "Carlos";
  document.getElementById("mensagem").innerText = "Bem-vindo, " + nome + "!";
</script>

Entrada de Dados

  1. prompt
    • Solicita ao usuário uma entrada de dados por meio de uma janela pop-up.
    • Sintaxe: let variavel = prompt("Digite seu nome:");
    • Exemplo
      JavaScript
      
let idade = prompt("Qual a sua idade?");
console.log("Idade do usuário: " + idade);  // Exibe a idade informada no console
  2. Campo input HTML
    • Permite que o usuário insira dados diretamente em um campo de texto na página.
    • Requer o uso de um botão para acionar o código que processa a entrada.
    • Exemplo
      JavaScript
      
<!DOCTYPE html>;
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Entrada de Dados com Input</title>
</head>
<body>
    <h1>Entrada de Dados com Input e Botão</h1>

    <!-- Campo de entrada de dados e botão para enviar -->
    <label for="nomeInput">Digite seu nome:</label>
    <input type="text" id="nomeInput" placeholder="Digite seu nome aqui">
    <button onclick="exibirNome()">Enviar</button>

    <!-- Parágrafo onde o nome será exibido -->
    <p id="saida"></p>

    <script>
    function exibirNome() {
        // Obtém o valor do campo de entrada de dados
        let nome = document.getElementById("nomeInput").value;

        // Exibe o nome no parágrafo com id 'saida'
        document.getElementById("saida").innerText = "Olá, " + nome + "!";
    }
    </script>
</body>
</html>

Exercício Prático

Criar uma página onde:

  • O usuário é solicitado a informar seu nome e sua cor favorita usando prompt.
  • A página então exibe uma mensagem de saudação com o nome e altera o fundo da página para a cor favorita do usuário.
