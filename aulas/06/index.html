Laços de Repetição em JavaScript

Nessa aula, o foco é ensinar o conceito de iteração e como realizar repetições no código de maneira eficiente usando os laços for, while e do...while.

Objetivo da Aula

  • Compreender o conceito de laços de repetição.
  • Aprender a utilizar os laços for, while e do...while para executar blocos de código repetidamente.
  • Identificar o laço adequado para diferentes situações.

Conteúdo da Aula

  1. Introdução aos Laços de Repetição

    Laços de repetição são usados para executar um bloco de código várias vezes. Dependendo da estrutura de laço utilizada, o número de repetições pode ser baseado em uma condição ou uma contagem específica.

  2. Estrutura de Repetição for

    O laço for é ideal para quando sabemos quantas vezes o bloco de código precisa ser repetido. A estrutura é composta por três partes: inicialização, condição e incremento.

    • Sintaxe e Exemplo
      JavaScript
      
for (let i = 1; i <= 5; i++) {
    console.log("Contagem: " + i);
}

  3. Estrutura de Repetição while

    O laço while é usado quando não sabemos previamente quantas vezes o bloco será repetido, mas sabemos uma condição que precisa ser verdadeira para continuar repetindo.

    • Sintaxe e Exemplo
      JavaScript
      
let contador = 1;

while (contador <= 5) {
    console.log("Contagem: " + contador);
    contador++;
}

  4. Estrutura de Repetição do...while

    O laço do...while funciona de forma semelhante ao while, mas garante que o bloco de código seja executado pelo menos uma vez antes de verificar a condição.

    • Sintaxe e Exemplo
      JavaScript
      
let numero = 1;

do {
    console.log("Número: " + numero);
    numero++;
} while (numero <= 5);

Neste exemplo, o bloco de código é executado uma vez antes de verificar se numero <= 5, o que garante que o valor inicial é exibido ao menos uma vez.

Exemplos Práticos

  1. Contando de 1 a 10 com for

    JavaScript
    
for (let i = 1; i <= 10; i++) {
    console.log("Número: " + i);
}

  2. Perguntar ao usuário até que ele digite "sair" com while

    JavaScript
    
let entrada = prompt("Digite algo (ou 'sair' para parar):");

while (entrada !== "sair") {
    console.log("Você digitou: " + entrada);
    entrada = prompt("Digite algo (ou 'sair' para parar):");
}

  3. Executando ao menos uma vez com do...while

    JavaScript
    
let valor;

do {
    valor = prompt("Digite um número maior que 10:");
} while (valor <= 10);

console.log("Número válido: " + valor);

Exercício Prático

Criar uma página onde:

  • O usuário insere um número entre 1 e 10 através do prompt.
  • Exiba no console a tabuada de multiplicação desse número de 1 a 10.
  • Use o laço for para a tabuada.
