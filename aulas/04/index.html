Operadores e Expressões em JavaScript

Nesta aula, os alunos aprenderão sobre diferentes operadores e como usá-los para criar expressões em JavaScript.

Introdução e Objetivos

  • Entender e utilizar operadores aritméticos, lógicos, comparativos e ternário.
  • Praticar a criação de expressões e condições usando diferentes operadores.

Conceitos e Explicações

  1. Os operadores aritméticos permitem realizar operações matemáticas básicas.
    • Adição (+): soma de dois valores.
    • Subtração (-): subtrai um valor do outro.
    • Multiplicação (*): multiplica dois valores.
    • Divisão (/): divide um valor pelo outro.
    • Módulo (%): retorna o resto da divisão.
    • Incremento (++) e Decremento (--): incrementa ou decrementa o valor de uma variável em 1.
    • Exemplo de Operadores Aritméticos
      JavaScript
      
let a = 10;
let b = 3;

console.log("Adição: " + (a + b));        // Resultado: 13
console.log("Subtração: " + (a - b));     // Resultado: 7
console.log("Multiplicação: " + (a * b)); // Resultado: 30
console.log("Divisão: " + (a / b));       // Resultado: 3.3333...
console.log("Módulo: " + (a % b));        // Resultado: 1
a++; // Incrementa a
console.log("Incremento de a: " + a);     // Resultado: 11
b--; // Decrementa b
console.log("Decremento de b: " + b);     // Resultado: 2

Operadores Comparativos

  1. Os operadores comparativos avaliam a relação entre dois valores, resultando em true ou false.
    • Igualdade (==) e Identidade (===): verifica se os valores são iguais. A identidade compara também o tipo de dado.
    • Diferente (!=) e Estritamente Diferente (!==): verifica se os valores são diferentes. O estritamente diferente compara o tipo de dado.
    • Maior que (>) e Maior ou Igual (>=): verifica se um valor é maior ou igual a outro.
    • Menor que (<) e Menor ou Igual (<=): verifica se um valor é menor ou igual a outro.
    • Exemplo
      JavaScript
      
let x = 5;
let y = "5";

console.log(x == y);   // true - valores iguais, mas tipos diferentes
console.log(x === y);  // false - valores iguais, mas tipos diferentes
console.log(x != y);   // false - valores iguais
console.log(x !== y);  // true - tipos diferentes
console.log(x > 3);    // true - x é maior que 3
console.log(x <= 5);   // true - x é igual a 5

Operadores Lógicos

  1. Os operadores lógicos são utilizados para combinar condições.
    • E (&&): verdadeiro se todas as condições forem verdadeiras.
    • Ou (||): verdadeiro se ao menos uma condição for verdadeira.
    • Não (!): inverte o valor lógico (se true vira false e vice-versa).
    • Exemplo
      JavaScript
      
let idade = 20;
let temCarteira = true;

console.log(idade >= 18 && temCarteira);  // true - idade maior ou igual a 18 e tem carteira
console.log(idade < 18 || temCarteira);   // true - ao menos uma condição é verdadeira
console.log(!temCarteira);                // false - inverte o valor de temCarteira

Operador Ternário

  1. O operador ternário permite escrever uma condição simplificada.
    • Sintaxe: condicao ? valorSeVerdadeiro : valorSeFalso
    • Exemplo
      JavaScript
      
let nota = 7.5;
let resultado = nota >= 6 ? "Aprovado" : "Reprovado";
console.log(resultado);  // "Aprovado" se nota >= 6, caso contrário "Reprovado"

Exercício Prático

Criar uma página onde:

  • O usuário insere a idade e sua altura usando prompt.
  • Use operadores comparativos e lógicos para verificar se:
    • A idade é maior ou igual a 18 e a altura é maior ou igual a 1.60.
  • Exiba no console.log se a pessoa pode ou não entrar em uma atração (ou “Passeio” ou “Montanha Russa”).
  • Use o operador ternário para atribuir uma mensagem de "Aprovado" ou "Reprovado" com base na condição.
