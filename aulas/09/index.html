Introdução ao DOM: Estrutura do DOM, Propriedades e Métodos Essenciais

Objetivos da Aula

Introduzir o DOM (Document Object Model), explicar sua estrutura hierárquica e demonstrar propriedades e métodos essenciais para manipulação de elementos de uma página web.

Conteúdo da Aula

  1. O que é o DOM?

    • O DOM (Document Object Model) é uma interface que permite interagir com o HTML de uma página através do JavaScript.
    • Ele representa o documento como uma estrutura hierárquica de nós (nodes), onde cada elemento HTML, atributo ou texto é um nó na árvore.
    • Exemplo: Estrutura do DOM
      JavaScript
      
<!DOCTYPE html>
<html>
<head>
    <title>Exemplo DOM</title>
</head>
<body>
    <h1>Bem-vindo!</h1>
    <p>Este é um parágrafo de exemplo.</p>
</body>
</html>
    • Estrutura da árvore DOM:
      JavaScript
      
Document
├── html
        ├── head
        │     └── title
        └── body
            ├── h1
            └── p

  2. Propriedades Essenciais do DOM

    Propriedades são usadas para acessar ou modificar elementos e atributos na árvore DOM.

    Propriedade Descrição
    document Representa todo o documento HTML carregado.
    document.body Acessa o elemento da página.
    document.title Obtém ou define o título da página.
    parentNode Retorna o nó pai de um elemento.
    childNodes Retorna todos os nós filhos de um elemento.
    firstChild Obtém o primeiro nó filho de um elemento.
    lastChild Obtém o último nó filho de um elemento.

    Exemplo: Usando propriedades

    JavaScript
    
let header = document.getElementById("meuTitulo"); // Busca pelo ID
let paragrafo = document.querySelector("p"); // Seleciona o primeiro 
    
console.log(header, paragrafo);

Exercícios Práticos

  1. Exemplo: Acessando elementos no DOM

    Dado o HTML abaixo:

    HTML
    
<!DOCTYPE html>
<html>
<head>
    <title>Teste DOM</title>
</head>
<body>
    <h1 id="titulo">Olá Mundo!</h1>
    <p class="texto">Primeiro parágrafo.</p>
    <p class="texto">Segundo parágrafo.</p>
</body>
</html>

    Código JavaScript:

    JavaScript
    
// Selecionando o título pelo ID
let titulo = document.getElementById("titulo");
console.log(titulo.innerText); // Exibe "Olá Mundo!"

// Selecionando todos os parágrafos pela classe
let paragrafos = document.querySelectorAll(".texto");
paragrafos.forEach((p, index) => {
    console.log(`Parágrafo ${index + 1}: ${p.innerText}`);
});
  2. Exemplo: Alterando elementos do DOM

    Dado o HTML abaixo:

    HTML
    
<!DOCTYPE html>
<html>
<body>
    <h1 id="cabecalho">Bem-vindo!</h1>
    <p>Este texto será alterado.</p>
</body>
</html>

    Código JavaScript:

    JavaScript
    
let cabecalho = document.getElementById("cabecalho");
cabecalho.innerText = "Novo Título"; // Altera o texto do título

let paragrafo = document.querySelector("p");
paragrafo.style.color = "blue"; // Altera a cor do texto para azul
  3. Exemplo: Criando e removendo elementos

    Dado o HTML abaixo:

    HTML
    
<!DOCTYPE html>
<html>
<body>
    <ul id="lista">
        <li>Item 1</li>
        <li>Item 2</li>
    </ul>
</body>
</html>

    Código JavaScript:

    JavaScript
    
let lista = document.getElementById("lista");

// Criando um novo elemento
let novoItem = document.createElement("li");
novoItem.innerText = "Item 3";
lista.appendChild(novoItem); // Adiciona o novo item à lista

// Removendo o primeiro item
let primeiroItem = lista.firstChild;
lista.removeChild(primeiroItem);
