Selecionando Elementos no DOM

Objetivos da Aula

  • Entender como selecionar elementos no DOM utilizando métodos JavaScript.
  • Diferenças entre métodos como getElementById, querySelector, e outros.

Conteúdo da Aula

  1. Selecionando um elemento pelo ID

    • Exemplo de Referência:
      JavaScript
      
let element = document.getElementById("myId");
console.log(element);
    • Exemplo Prático: Altere o texto de um elemento com id="title".
      JavaScript
      
<h1 id="title">Texto Original</h1>
<script>
    let element = document.getElementById("title");
    element.textContent = "Novo Texto";
</script>

  2. Selecionando elementos por classe

    • Exemplo de Referência:
      JavaScript
      
let elements = document.getElementsByClassName("myClass");
console.log(elements);
    • Exemplo Prático: Altere o estilo de todos os elementos com a classe highlight.
      JavaScript
      
<p class="highlight">Primeiro Parágrafo</p>
<p class="highlight">Segundo Parágrafo</p>
<script>
    let elements = document.getElementsByClassName("highlight");
    for (let element of elements) {
    element.style.color = "blue";
    }
</script>

  3. Selecionando elementos por tag

    • Exemplo de Referência:
      JavaScript
      
let elements = document.getElementsByTagName("p");
console.log(elements);

  4. Selecionando um elemento com querySelector

    • Exemplo de Referência:
      JavaScript
      
let element = document.querySelector(".myClass");
console.log(element);

  5. Selecionando múltiplos elementos com querySelectorAll

    • Exemplo de Referência:
      JavaScript
      
let elements = document.querySelectorAll(".myClass");
console.log(elements);

Exercício

Manipulalção básica do DOM

  1. Crie um botão que altera o texto de um elemento com ID específico ao ser clicado.
  2. Altere o estilo de todos os elementos de uma classe.
  3. Liste no console todos os elementos <li> de uma lista.
