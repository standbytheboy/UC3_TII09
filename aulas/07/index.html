Funções em JavaScript

Declaração, parâmetros, retorno, funções anônimas e arrow functions.

Objetivos da Aula

  • Compreender o conceito e a utilidade das funções em JavaScript.
  • Aprender a declarar funções e trabalhar com parâmetros e retorno.
  • Introduzir funções anônimas e arrow functions.
  • Mostrar a diferença entre os tipos de funções.

Conteúdo da Aula

  1. O que são Funções?

    Blocos de código reutilizáveis que executam uma tarefa específica. Melhoram a organização, reduzem a repetição e tornam o código mais legível.

  2. Declaração de Funções

    • Sintaxe básica:
      JavaScript
      
function nomeDaFuncao(parametros) {
    // Código a ser executado
    return resultado;
}
    • Exemplo:
      JavaScript
      
function saudacao(nome) {
    return "Olá, " + nome + "!";
}

console.log(saudacao("João"));  // Saída: Olá, João!

  3. Funções com Parâmetros

    Parâmetros permitem que uma função receba valores para trabalhar.

    • Exemplo com cálculo:
      JavaScript
      
function somar(a, b) {
    return a + b;
}

console.log(somar(5, 3));  // Saída: 8

  4. Funções com Retorno

    Funções podem retornar valores usando a palavra-chave return. Após o retorno, a execução da função é encerrada.

    • Exemplo:
      JavaScript
      
function verificarParidade(numero) {
    if (numero % 2 === 0) {
        return "Par";
    } else {
        return "Ímpar";
    }
}

console.log(verificarParidade(4));  // Saída: Par

  5. Funções Anônimas

    Funções sem nome, geralmente usadas em variáveis ou eventos.

    • Exemplo:
      JavaScript
      
const multiplicar = function(a, b) {
    return a * b;
};

console.log(multiplicar(3, 4));  // Saída: 12

  6. Arrow Functions

    Forma mais curta de declarar funções, introduzida no ES6. Não possuem seu próprio this e não são adequadas para métodos de objetos.

    • Sintaxe:
      JavaScript
      
const nomeDaFuncao = (parametros) => expressao;
    • Exemplo:
      JavaScript
      
const dividir = (a, b) => a / b;
console.log(dividir(10, 2));  // Saída: 5

Exercício Prático

Criar funções simples e compreender sua execução.

  • Criar uma Função de Saudação
    1. Escreva uma função chamada saudacao que recebe um nome como parâmetro e retorna uma mensagem de boas-vindas.
    2. Exiba o resultado no console.
  • Criar uma Função de Calculadora
    1. Escreva uma função chamada calculadora que receba dois números e uma operação (+, -, *, /) como parâmetros.
    2. Retorne o resultado da operação ou uma mensagem de erro para operações inválidas.
