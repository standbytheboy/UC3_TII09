Variáveis e Tipos de Dados.

Declaração de variáveis, tipos de dados primitivos e operações básicas em JavaScript.
Objetivo: Ensinar os conceitos fundamentais de variáveis e tipos de dados em JavaScript, além de explorar operações básicas como soma, concatenação e comparação entre valores.

Introdução e Objetivos

  • Explicar o que são variáveis e por que elas são essenciais para armazenar e manipular dados.
  • Introduzir tipos de dados primitivos e operações básicas.
  • Preparar os alunos para manipular variáveis de forma prática, aplicando-as em operações comuns no desenvolvimento web.

Conceitos e Explicações

  1. let, const, e var: Explicação de cada palavra-chave.
    • let: Para variáveis que podem ser alteradas.
    • const: Para variáveis cujo valor não muda.
    • var: Antiga forma de declarar variáveis, geralmente evitada em projetos modernos.
    • Exemplo
      JavaScript
      
let idade = 25;
const nome = "Aecio";
var cidade = "São Paulo";

Tipos de Dados Primitivos

  1. Número: Inclui inteiros e decimais.
    • Exemplo: let preco = 19.99;
  2. String: Conjunto de caracteres, como textos.
    • Exemplo: let saudacao = "Olá, mundo!";
  3. Booleano: Valores lógicos (true ou false).
    • Exemplo: let estaAtivo = true;
  4. Undefined e Null:
    • Undefined representa uma variável sem valor atribuído.
    • Null é intencionalmente vazia.
    • Exemplo
      JavaScript
      
let valor;  // Undefined
let vazio = null;  // Null

Operações Básicas

  1. Aritméticas (soma, subtração, multiplicação, divisão):
    • Exemplo
      JavaScript
      
let a = 10;
let b = 5;
let soma = a + b;  // Resultado: 15
let subtracao = a - b;  // Resultado: 5
  2. Concatenação de Strings:
    • Exemplo
      JavaScript
      
let nome = "Maria";
let saudacao = "Olá, " + nome + "!";  // Resultado: "Olá, Maria!"
  3. Comparação (igualdade, desigualdade):
    • Exemplo
      JavaScript
      
let x = 10;
let y = "10";
console.log(x == y);  // true (compara o valor, não o tipo)
console.log(x === y);  // false (compara o valor e o tipo)
