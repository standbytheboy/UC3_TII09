Estruturas Condicionais em JavaScript

Nessa aula, o foco é entender como controlar o fluxo de um programa usando estruturas condicionais.

Objetivo da Aula

  • Compreender e implementar estruturas condicionais if, else if, else e switch.
  • Criar condições mais complexas para controlar o fluxo da aplicação.

Conteúdo da Aula

  1. Estrutura Condicional if, else if e else

    A estrutura if, else if e else permite que o código execute blocos específicos com base em uma condição.

    • if: executa o bloco de código se a condição for verdadeira.
    • else if: opcional, verifica uma nova condição se a anterior for falsa.
    • else: opcional, executa o bloco de código caso todas as condições anteriores sejam falsas.
    • Sintaxe e Exemplo
      JavaScript
      
let nota = parseInt(prompt("Digite a nota do aluno:"));

if (nota >= 7) {
    console.log("Aprovado");
} else if (nota >= 5) {
    console.log("Recuperação");
} else {
    console.log("Reprovado");
}

  2. Estrutura Condicional switch

    A estrutura switch é usada para comparar o valor de uma variável a várias opções (case). É útil para substituir múltiplas condições if quando se tem uma variável com valores específicos e previsíveis.

    • case: compara o valor da variável.
    • break: interrompe a execução do bloco switch (se não utilizar, os blocos seguintes também serão executados).
    • default: executa caso nenhum case corresponda ao valor da variável.
    • Sintaxe e Exemplo
      JavaScript
      
let diaSemana = parseInt(prompt("Digite um número de 1 a 7 para o dia da semana:"));

switch (diaSemana) {
    case 1:
        console.log("Domingo");
        break;
    case 2:
        console.log("Segunda-feira");
        break;
    case 3:
        console.log("Terça-feira");
        break;
    case 4:
        console.log("Quarta-feira");
        break;
    case 5:
        console.log("Quinta-feira");
        break;
    case 6:
        console.log("Sexta-feira");
        break;
    case 7:
        console.log("Sábado");
        break;
    default:
        console.log("Número inválido. Digite de 1 a 7.");
}

Neste exemplo, cada case representa um dia da semana, e a estrutura switch exibe o dia correspondente ao número inserido.

Exemplos Práticos

  1. Usando if para Verificar Idade

    JavaScript
    
let idade = parseInt(prompt("Digite a sua idade:"));

if (idade < 18) {
    console.log("Menor de idade");
} else if (idade >= 18 && idade < 65) {
    console.log("Maior de idade");
} else {
    console.log("Idoso");
}

  2. Usando switch para Exibir uma Mensagem Baseada em uma Nota

    JavaScript
    
let nota = parseInt(prompt("Digite a nota final (de 0 a 10):"));

switch (nota) {
    case 10:
    case 9:
        console.log("Excelente!");
        break;
    case 8:
    case 7:
        console.log("Bom");
        break;
    case 6:
    case 5:
        console.log("Regular");
        break;
    default:
        console.log("Insuficiente");
}

Exercício Prático

Criar uma página onde:

  • O usuário insere a sua idade.
  • Se a idade for menor que 12, exiba a mensagem “Criança”.
  • Se a idade estiver entre 12 e 17, exiba “Adolescente”.
  • Se a idade estiver entre 18 e 64, exiba “Adulto”.
  • Se a idade for 65 ou mais, exiba “Idoso”.

Use if, else if e else para realizar essas condições e exiba o resultado no console.log.
