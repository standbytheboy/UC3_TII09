Manipulação do DOM

A manipulação do Document Object Model (DOM) é uma técnica que permite alterar o conteúdo de páginas web sem que seja necessário recarregar a página.

Contempla:

  • Selecionar e manipular elementos
  • Adicionar e remover elementos
  • Alterar estilos, textos e atributos dos elementos

Buscando Elementos

  1. Busca por Id

    JavaScript
    
// buscando por id
const element = document.querySelector('#elementId');

// buscando diversos elementos por id (ignora o propósito de um identificador único)
const elements = document.querySelectorAll('#elementId');

  2. Busca por classe

    JavaScript
    
// buscar elemento pelo nome da classe
const element = document.querySelector('.className');

// buscar grupo de elementos pelo nome da classe
const elements = document.querySelectorAll('.className');

  3. Exemplo 2:

    JavaScript
    
li.remove();

Criando Elementos

  1. Síntaxe

    JavaScript
    
const element = document.createElement('element');
parentElement.appendChild(element);

  2. Exemplo 1:

    JavaScript
    
const div = document.createElement('div');
body.appendChild(div);

  3. Exemplo 2:

    JavaScript
    
const ul = document.createElement('ul');
const li = document.createElement('li');
ul.appendChild(li);

Removendo Elementos

  1. Síntaxe

    JavaScript
    
// com a função removeChild
parent.removeChild(child);

//  com a função remove
child.remove();

  2. Exemplo 1:

    JavaScript
    
ul.removeChild(li);

  3. Exemplo 2:

    JavaScript
    
li.remove();

Modificando Elementos

  1. Modificando o estilo

    JavaScript
    
element.style.fontSize = "18px";
element.style.backgroundColor = "#ffffff";

  2. Modificando o Id

    JavaScript
    
element.id = "myId";

  3. Modificando a classe

    JavaScript
    
element.classList.add('myClass');
element.classList.remove('myClass');

  4. Modificando o texto

    JavaScript
    
element.innerText = "Hello World";

  5. Modificando o atributo como um todo

    JavaScript
    
// Síntaxe
element.setAttribute(attribute, value);
element.removeAttribute(attribute);

// Examplo
input.setAttribute('name', 'myInput');
input.removeAttribute('name');
